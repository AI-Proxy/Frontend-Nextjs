# Markdown File Showcase

This Markdown file demonstrates how to use various features of Markdown.

## Headings

You can create headings using the hash symbol (#):

# Heading level 1

## Heading level 2

### Heading level 3

## Emphasis

To emphasize text, you can use the asterisk (\*) or underscore (\_):

_Italic text_
_Italic text_

**Bold text**
**Bold text**

## Lists

### Ordered List

1. First item
2. Second item
3. Third item

### Unordered List

-   Item 1
-   Item 2
-   Item 3

## Links

Creating links is easy. Just use the following syntax:

[Google](https://www.google.com)

## Images

You can also add images to your Markdown files:

![Markdown Logo](https://commonmark.org/images/markdown-mark.png)

## Blockquotes

> This is a blockquote.
> You can quote someone and highlight important information.

## Code Blocks

To display code snippets or blocks, use backticks (`) or triple backticks (```):

Inline code snippet: `console.log("Hello, World!");`

```javascript
// Multiline code block
function greet(name) {
    console.log("Hello, " + name + "!");
}
```

## Tables

You can create tables using pipes (|) to separate columns:

| Name    | Age | Gender |
| ------- | --- | ------ |
| John    | 25  | Male   |
| Jane    | 28  | Female |
| Michael | 30  | Male   |

## Horizontal Rule

You can add a horizontal rule using three or more hyphens (---), asterisks (\*\*\*), or underscores (\_\_\_):

---
